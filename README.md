wait.js
=======

Usage:

```
<script src = "wait.js"></script>

<script>

  wait.show();
  wait.hide();
  
</script>
```
Set <body> margin to 0 (recommended).

Features:
 
 - Customization Friendly.
 - Shows Time Elapsed.
 - Everything in one JS File

Requirement: 
 
 - jQuery

Config:

 - Default message is "Please Wait"
